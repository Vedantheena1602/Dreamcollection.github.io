# Dream Collection

A beautiful, modern website celebrating dreams, aspirations, and the journey to achievement.

## 🌟 Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and elegant interface with smooth animations
- **Multiple Pages**: Home, Dreams Collection, About Us, and Contact sections
- **Contact Form**: Functional contact form for community engagement
- **Social Links**: Easy integration with social media platforms

## 📄 Pages

- **index.html** - Home page with featured collections and introduction
- **dreams.html** - Explore different types of dreams and aspirations
- **about.html** - Learn about our mission, vision, and values
- **contact.html** - Get in touch with us via contact form
- **styles.css** - All styling and responsive design

## 🎨 Design Highlights

- Beautiful gradient color scheme (Indigo to Purple)
- Smooth hover effects and animations
- Card-based layout for content organization
- Mobile-first responsive design
- Professional typography and spacing

## 🌐 Domain

This website is configured with the custom domain: **dreamcollection.com**

## 🚀 Getting Started

1. Clone or download this repository
2. Open `index.html` in your browser (or visit https://dreamcollection.com)
3. Navigate through the pages using the navigation menu
4. Customize the content, colors, and information as needed

## 🔧 Customization

To customize the website:
- Edit the color scheme in `styles.css` (look for `:root` variables)
- Update content in HTML files
- Modify navigation links in the navbar
- Add your own images and social media links
